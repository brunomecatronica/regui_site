HiveMQ (público) • Dados em tempo real • Comandos bidirecionais

App Flutter

Monitoramento em tempo real, controle remoto, interface intuitiva

Sistema de Irrigação Automática

Sensor de Solo

Monitora umidade continuamente (0-100%)

Lógica Automática

Liga bomba se < 10%, desliga se >= 90%

Bomba Pulsada

1s ligada / 3s desligada (modo eficiente)

Plantas Irrigadas

Solo sempre na umidade ideal

Características Técnicas

FreeRTOS: 11 tasks paralelas rodando em 2 cores (Core 0: sistema, Core 1: aplicação)

Thread-Safe: Mutexes protegem acesso concorrente aos estados compartilhados

Persistência: Estados salvos na memória flash (NVS) - configurações mantidas após reinicialização

Detecção de Sensores: Sistema detecta automaticamente sensores desconectados e age com segurança

Tecnologias Utilizadas

ESP32

Microcontrolador de 32 bits com Wi-Fi integrado, dual-core, FreeRTOS para multi-tarefa real-time

MQTT

Protocolo leve para IoT, comunicação bidirecional em tempo real entre ESP32 e aplicativo

Flutter

Framework multiplataforma para aplicativo móvel com interface moderna e responsiva

FreeRTOS

Sistema operacional em tempo real para execução paralela de múltiplas tarefas

Sensores Analógicos

Sensor de umidade de solo, sensor de nível de reservatório, DHT11 (temperatura/umidade ar)

NVS (Non-Volatile Storage)

Armazenamento persistente para configurações, estados e credenciais Wi-Fi

Diferenciais Técnicos

  • Modo Pulsado: Bomba opera em ciclos de 1s ligada / 3s desligada, prolongando vida útil e economizando energia
  • Histerese Inteligente: LEDs usam histerese para evitar pisca constante, com zonas estáveis de transição
  • Health Check LCD: Sistema verifica e reinicializa automaticamente o display se detectar problemas
  • Thread-Safe: Todos os estados compartilhados protegidos por mutexes para acesso seguro entre tasks
  • Calibração Flexível: Valores ADC configuráveis para diferentes tipos de solo e reservatórios
